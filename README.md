# Treasure Hunt

## How To Run

```sh
$ make play
```
or

```sh
$ go run main.go
```

If you haven't install golang please [install](https://golang.org/doc/install) first.

## How To Play

How to Play :
- Find the treasure ($)
- Press "w" to move player (X) up
- Press "s" to move player (X) down
- Press "d" to move player (X) right
- Press "r" to reset the game
